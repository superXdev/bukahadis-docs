# Pengenalan BukaHadis API

Di api **BukaHadis**, kamu bisa mengambil berbagai teks hadis lengkap dengan tulisan arabnya dari 9 perawi: 
* Abu Daud
* Ahmad
* Bukhari
* Darimi
* Ibnu Majah
* Malik
* Muslim
* Nasai
* Tirmidzi

Semua response API di kirim dalam format JSON.
