
# Keyword Parameter

Mencari hadis-hadis berdasarkan kata kunci

## Request API

di tambah dengan parameter field untuk menampilkan judul dan terjemahannya saja

```shell
https://api.devinesia.com/buka-hadis/?keyword=sedekah&field=title,translated
```
